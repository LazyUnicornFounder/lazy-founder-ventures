## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Hosting:** Lovable
- **Build Tool:** Vite

## Getting Started

```bash
git clone https://github.com/LazyUnicornFounder/lazy-founder-ventures.git
cd lazy-founder-ventures
npm install
npm run dev
```
